# Iris-Flower-Classification
Data Science project for Iris Flower Classification using Python, Scikit-learn

This is a Data science project that classifies iris flowers into different species using the K-Nearest Neighbors (KNN) algorithm.


The Iris dataset contains flower measurements such as:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The model predicts the flower species:
- Setosa
- Versicolor
- Virginica

## 🛠 Technologies Used
- Python
- Scikit-learn
- NumPy
- Pandas
- Google Colab

## 📂 Dataset
This project uses the built-in Iris dataset from Scikit-learn.

## 🚀 Steps Performed
1. Loaded the Iris dataset
2. Split the dataset into training and testing sets
3. Trained the KNN classifier
4. Predicted flower species
5. Evaluated model accuracy

## 📊 Output
The model achieved high accuracy in classifying iris flowers.

## ▶️ How to Run
1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the notebook in Google Colab or Jupyter Notebook.
